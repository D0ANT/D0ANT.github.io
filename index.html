Bootstrap 4를 이용한 웹 템플릿

Grayscale은 Start Bootstrap에서 만든 무료 Bootstrap 테마입니다. 미리보기 페이지 에서 템플릿을 다운로드하기 만하면 됩니다 . 테마는 오픈 소스이며 개인 또는 상업적 목적으로 사용할 수 있습니다.

해안선 (사진)

Grayscale은 오픈 소스이며 MIT 라이센스가 있습니다. 근데 심심해서 그냥 넣어봄(원래 있었음)

미스티 (카트 클랜 미스틱 아님)

설명과 함께 프로젝트의 이미지 또는 기타 항목을 넣을 수있는 위치의 예입니다.
